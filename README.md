# GoCN Education

## Prerequisite

- Ruby version: `2.2.0`
- Jekyll version: `3.3.1`
- Node.js version: `4.7.1`

## Usage

```bash
make install
```

### Development

```bash
make serve
```

### Production

```bash
make build
```

## Collaboration

Travis CI will build `master` branch and run `make deploy`, which would push the build result to `release` branch.
