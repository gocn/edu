# GoCN Education

## Prerequisite

- Ruby version: `2.0.0` or above.
- Jekyll version: `3.3.1`

## Usage

### Development

```ruby
jekyll serve
```

### Production

```ruby
jekyll build
```

## Collaboration

Travis CI will build `master` branch and run `bash ./deploy.sh`, which would push the build result to `release` branch.
